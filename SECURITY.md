# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the OpenDecree TypeScript SDK, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please [report it via GitHub Security Advisories](https://github.com/opendecree/decree-typescript/security/advisories/new).

You should receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Scope

This policy covers the `@opendecree/sdk` package published on npm.
